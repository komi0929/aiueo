# 自動記録ワークフロー

コード変更が完了したら、以下を自動実行:

1. CHANGELOG.md に変更内容を追記する（日付・変更カテゴリ・内容）

2. 変更をステージングする

```
git add -A
```

3. コミットする（コミットメッセージは変更内容を簡潔に日本語で記述、絵文字プレフィックス付き）

```
git commit -m "✨ [変更の説明]"
```

絵文字プレフィックスの規則:

- ✨ 新機能
- 🐛 バグ修正
- 🎨 デザイン/UI変更
- ♻️ リファクタリング
- 🗑️ 削除/クリーンアップ
- 📝 ドキュメント
- 🔧 設定変更

4. GitHubにプッシュする

```
git push origin main
```
